sources/opentype/README.txt
==================

This file describes the fea source files included with the Kedebideri
font family. This information should be distributed along with the Kedebideri
fonts and any derivative works.

These files are part of the Kedebideri font family 
(https://software.sil.org/kedebideri/) and are

with Reserved Font Names "Kedebideri" and "SIL".

This Font Software is licensed under the SIL Open Font License,
Version 1.1.

All files utilize FEA extensions provided by pysilfont.

main.feax     Main source file for OpenType logic.
gpos.feax     Included into main.feax source file.
gsub.feax     Included into main.feax source file.
classes.txt		Manually created classes.
